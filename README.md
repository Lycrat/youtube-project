### A website for all things Vagrant Holiday related. Hosts all videos of Vagrant Holiday and allow fellow vagrants to upload their own videos.
Including the usage of:
- Express Sessions
- SQL
- Bootstrap

Features:
- Video Uploading
- Video Streaming
